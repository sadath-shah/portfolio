Student Registration System

✅ Student Registration System Overview Purpose: Web-based application for managing student data in a structured and user-friendly way. ✅ Technologies Used: Backend: PHP, MySQL Frontend: HTML, CSS, Bootstrap, JavaScript ✅ Core Features: Student Registration Form: Clean, responsive layout using Bootstrap. Fields: First Name, Last Name, Email, Contact Number, Course, Profile Picture. Client-side validation with HTML5 and JavaScript. Server-side validation for security: Duplicate entry checks. Proper data formatting. Passwords hashed using PHP’s password_hash() function. Profile image upload: Saved in a dedicated server folder. Image path stored in MySQL alongside student data. Admin Panel: Login-protected using session management. Dashboard displaying all registered students in a table format. Admin Actions: View complete student profiles (including profile pictures). Edit student information with pre-filled forms. Delete student records permanently. Responsive design for both desktop and mobile. ✅ Backend Concepts Demonstrated: Session management for admin authentication. Database CRUD operations (Create, Read, Update, Delete). Secure password storage. Image handling and storage. Clean, responsive UI using Bootstrap.
